## 赛车模型参数

### 通用参数
所有赛车模型共享以下通用参数：

| 参数名 | 类型 | 默认值 | 说明 |
|--------|------|--------|------|
| name | SFString | "car_xxx" | 赛车名称 |
| translation | SFVec3f | 0 0 0.302 | 赛车位置 (x, y, z) |
| rotation | SFRotation | 0 0 1 0 | 赛车旋转 (x, y, z, angle) |
| controller | SFString | "car" | 控制器名称 |
| customData | SFString | "" | 自定义数据 |
| bodySize | SFVec3f | 4.2 1.8 0.12 | 车身尺寸 (长, 宽, 高) |
| mass | SFFloat | 800.0 | 车辆质量 (kg) |

### 各车型详细参数

#### 1. CarFrost - 冰霜白极光车
- **车身颜色**: 0.95 0.95 0.97 (浅灰白色)
- **外观特征**: 白色车身，蓝色鼻锥装饰
- **材质**: roughness 0.3, metalness 0.4

#### 2. CarNova - 新星黄闪电车
- **车身颜色**: 1.0 0.84 0.0 (金黄色)
- **外观特征**: 黄色车身，橙色鼻锥装饰
- **材质**: roughness 0.3, metalness 0.4

#### 3. CarPhoenix - 烈焰红流线型
- **车身颜色**: 0.86 0.08 0.24 (深红色)
- **外观特征**: 红色车身，橙黄色鼻锥装饰
- **材质**: roughness 0.3, metalness 0.4

#### 4. CarShadow - 暗夜黑隐形车
- **车身颜色**: 0.15 0.15 0.15 (深黑色)
- **外观特征**: 黑色车身，深灰色鼻锥装饰
- **材质**: roughness 0.3, metalness 0.4

#### 5. CarThunder - 电光蓝肌肉车
- **车身颜色**: 0.12 0.56 1.0 (电光蓝)
- **外观特征**: 蓝色车身，宽体设计，蓝色宽体翼子板
- **材质**: roughness 0.4, metalness 0.3

#### 6. CarViper - 毒蛇绿极速车
- **车身颜色**: 0.13 0.55 0.13 (深绿色)
- **外观特征**: 绿色车身，绿色鼻锥装饰
- **材质**: roughness 0.3, metalness 0.4

## 参数说明

### 道路系统

#### Road（直道）
**节点结构**：
```proto
Road {
  translation x y z           # 道路中心位置
  rotation 0 0 1 angle        # 道路旋转角度 (弧度)
  name "road_xxx"             # 道路名称
  width 10                    # 道路宽度
  rightBarrier TRUE           # 右侧是否有护栏
  leftBarrier TRUE            # 左侧是否有护栏
  bottom TRUE                 # 底部是否有地面
  wayPoints [                 # 路径点列表
    x1 y1 z1
    x2 y2 z2
    ...
  ]
  splineSubdivision 8         # 曲线细分度 (仅曲线道路)
  appearance {                # 外观设置
    baseColor r g b           # 颜色 (0-1)
    roughness 0.9             # 粗糙度
    metalness 0               # 金属度
  }
}
```

#### CurvedRoadSegment（曲线段）

**节点结构**：
```proto
CurvedRoadSegment {
  translation x y z           # 曲线段中心位置
  rotation 0 0 1 angle        # 曲线段旋转角度
  name "curved_seg_x"         # 曲线段名称
  width 10                    # 道路宽度
  rightBarrier TRUE           # 右侧护栏
  leftBarrier TRUE            # 左侧护栏
  bottom TRUE                 # 底部地面
  curvatureRadius 10.5        # 曲率半径 (单位)
  appearance {                # 外观设置
    baseColor r g b           # 颜色
    roughness 0.9             # 粗糙度
    metalness 0               # 金属度
  }
}
```

### 检查点系统

#### Checkpoint（检查点）

**节点结构**：
```proto
Solid {
  translation x y z           # 检查点位置
  rotation 0 0 1 angle        # 检查点旋转角度
  name "checkpoint_x"         # 检查点名称
  children [
    Shape {
      appearance PBRAppearance {
        baseColor 0 0.8 1     # 颜色 (青蓝色)
        transparency 0.6      # 透明度 (0-1)
        roughness 0.3         # 粗糙度
        metalness 0           # 金属度
      }
      geometry Box {
        size 10 0.1 2         # 尺寸 (长, 宽, 高)
      }
    }
  ]
}
```
### 起跑线

#### Start/Finish Line（起跑线）

**节点结构**：
```proto
Solid {
  translation x y z           # 起跑线位置
  rotation 0 0 1 angle        # 起跑线旋转角度
  name "start_finish_line"    # 起跑线名称
  children [
    Shape {
      appearance PBRAppearance {
        baseColor 0.05 0.05 0.05  # 颜色 (深灰色)
        roughness 0.9             # 粗糙度
        metalness 0               # 金属度
      }
      geometry Box {
        size 0.5 10 0.001         # 尺寸 (长, 宽, 高)
      }
    }
  ]
}
```
### 发车信号灯

## 简单赛道 (track_basic)

### 赛道概述

简单赛道是一个矩形的闭环赛道，由4条直道和4个弯道组成。赛道宽度为10单位，共有4个检查点和1条起跑线。

## 开发注意事项

1. **坐标系说明**
   - Webots使用右手坐标系
   - X轴: 向右为正
   - Y轴: 向前为正
   - Z轴: 向上为正

2. **单位说明**
   - 所有距离单位为米
   - 所有角度单位为弧度
   - 时间单位为秒

3. **性能优化**
   - 合理设置仿真步长
   - 优化传感器采样频率
   - 减少不必要的计算

4. **调试技巧**
   - 使用Webots内置调试工具
   - 查看传感器数据可视化
   - 记录赛车轨迹和传感器数据

5. **测试建议**
   - 先测试单个功能模块
   - 逐步集成测试
   - 进行多轮仿真测试
   - 记录测试结果和问题

## 附录

### 信号灯控制流程
```
红灯 → 黄灯 → 绿灯 → 黄灯 → 红灯 → ...
  ↓       ↓       ↓       ↓       ↓
停止    准备    出发    准备    停止
```

### webots 赛道设计经验

- 明确大致布局后可生成设计文档，并让大模型根据文档生成赛道元素的3D模型。
- 尽可能使用少的元素数量，降低开发复杂度
- 手动微调元素的参数
- 最新版本的webots中道路用waypoint代替了length参数，在使用大模型生成时要格外注意
- 对于复用度高的节点可以提取为prototype ，方便后续修改和维护。
