# 汽车模型升级总结

## 升级概述
所有六辆汽车模型都已升级为增强版，添加了更美观的车身、车头、车窗等细节设计。

## 修复说明

### 已修复的问题
1. **bodySize字段错误** - 修复了PROTO参数'bodySize'没有匹配IS字段的错误
2. **摄像头配置** - 从单个前置摄像头升级为左右两个摄像头

### 修复详情
- **bodySize使用**: 现在在主车身和碰撞体中正确使用`IS bodySize`
- **摄像头配置**: 每辆车现在有两个摄像头（left_camera和right_camera）
  - 左摄像头位置: (2.1, -0.3, 0.08)，旋转角度: -0.1弧度
  - 右摄像头位置: (2.1, 0.3, 0.08)，旋转角度: 0.1弧度

## 新增特性详解

### 车身结构
1. **下层底盘**: 主要车身结构，使用`IS bodySize`参数
2. **上层车身**: 驾驶舱部分，长2.2m，宽1.4m，高0.35m
3. **前引擎盖**: 倾斜设计，增加空气动力学效果

### 车窗系统
1. **前挡风玻璃**: 倾斜角度-0.4弧度，半透明玻璃效果
2. **后窗**: 倾斜角度0.5弧度，半透明玻璃效果
3. **侧车窗**: 左右两侧，半透明玻璃效果
   - 颜色: 浅蓝色 (0.6 0.75 0.85)
   - 透明度: 0.3
   - 粗糙度: 0.05
   - 金属度: 0.1

### 灯光系统
1. **前大灯**:
   - 颜色: 白色 (1.0 1.0 0.9)
   - 发光颜色: (1.0 1.0 0.8)
   - 尺寸: 0.1 x 0.15 x 0.03m
   - 粗糙度: 0.1

2. **尾灯**:
   - 颜色: 红色 (1.0 0.1 0.1)
   - 发光颜色: (0.8 0.0 0.0)
   - 尺寸: 0.08 x 0.12 x 0.03m
   - 粗糙度: 0.1

### 摄像头系统（新增）
1. **左摄像头**:
   - 名称: `left_camera`
   - 位置: (2.1, -0.3, 0.08)
   - 旋转: (0, 0, 1, -0.1)
   - 分辨率: 640 x 480
   - 视野角: 1.047弧度 (60度)

2. **右摄像头**:
   - 名称: `right_camera`
   - 位置: (2.1, 0.3, 0.08)
   - 旋转: (0, 0, 1, 0.1)
   - 分辨率: 640 x 480
   - 视野角: 1.047弧度 (60度)

### 外观细节
1. **后视镜**: 左右各一个，深灰色金属质感
2. **后扰流板**: 增加下压力
3. **后翼**: 水平尾翼设计
4. **车门线条**: 黑色装饰线条
5. **赛车条纹**: 每辆车都有独特的颜色和设计

## 技术规格

### 物理属性
- **质量**: 800kg
- **质心**: [0, 0, -0.02]
- **密度**: -1 (使用质量而非密度)
- **车身尺寸**: 4.2 x 1.8 x 0.12m (可通过bodySize参数配置)

### 车轮系统
- **车轮数量**: 4个
- **车轮半径**: 0.3m
- **车轮宽度**: 0.25m
- **车轮颜色**: 深灰色 (0.1 0.1 0.1)
- **车轮粗糙度**: 0.8

### 传感器
1. **左摄像头**:
   - 分辨率: 640 x 480
   - 视野角: 1.047弧度 (60度)
   - 位置: 车头左侧，略微向左偏转

2. **右摄像头**:
   - 分辨率: 640 x 480
   - 视野角: 1.047弧度 (60度)
   - 位置: 车头右侧，略微向右偏转

3. **惯性测量单元 (IMU)**:
   - 位置: 车身中心
   - 用于测量加速度和角速度

### 电机系统
- **左后轮电机**: 最大速度40 rad/s
- **右后轮电机**: 最大速度40 rad/s
- **前轮**: 无动力，自由转向

## 升级效果

### 视觉改进
1. **更真实的汽车外观**: 添加了车窗、车门线条等细节
2. **更好的空气动力学**: 倾斜的引擎盖和挡风玻璃
3. **更丰富的细节**: 后视镜、扰流板、尾翼等
4. **独特的个性**: 每辆车都有独特的颜色和装饰

### 功能改进
1. **更好的视野**: 左右两个摄像头提供更广阔的视野
2. **更稳定的物理**: 质心调整
3. **更真实的灯光**: 发光的前大灯和尾灯
4. **可配置的车身**: bodySize参数允许自定义车身尺寸
