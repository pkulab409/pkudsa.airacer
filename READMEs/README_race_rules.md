# 赛事规则文档

## 1. 赛制结构

系统根据参赛队伍数量自动选择赛制（`server/race/bracket.py`）：

```
<= 4 队:  排位赛 -> 决赛
5-8 队:  排位赛 -> 半决赛 -> 决赛
>= 9 队:  排位赛 -> 小组赛 -> 半决赛 -> 决赛
```

| 阶段 | 圈数 | 每场最大车辆数 | 晋级规则 |
|------|------|-------------|---------|
| 排位赛 (placement) | 2 圈 | 4 辆 | 计时赛，用于蛇形分组种子排位；<=4 队时前 4 晋级，>=9 队时全部晋级 |
| 小组赛 (group_stage) | 3 圈 | 4 辆 | 各组第 1 名 + 所有第 2 名中完赛时间最短的 1 队 -> 晋级 |
| 半决赛 (semi) | 3 圈 | 4 辆 | 每场前 2 名晋级 |
| 决赛 (final) | 5 圈 | 4 辆 | 决出最终名次 |

场次数量由 `ceil(队伍数 / 4)` 自动计算，至少 1 场。

---

## 2. 排名规则

### 竞速赛（小组赛 / 半决赛 / 决赛）

1. 第一辆车完成规定圈数时，启动 **60 秒宽限期**（grace period）
2. 宽限期内完赛的车辆：按**完赛时间升序**排列
3. 宽限期结束时仍未完赛的车辆：按**已完成圈数降序**、**lap_progress 降序**排列

### 排位赛

- 每辆车完成 2 圈，取**最快单圈时间**作为**蛇形分组种子排位**依据
- 各场次独立仿真，互不影响
- 现场名次仍按**完赛总时间**排序（与竞速赛一致）

## 3. 碰撞与违规惩罚

| 判定 | 条件 | 惩罚 |
|------|------|------|
| 轻微碰撞 | 车辆间低速接触（相对速度 < 3 m/s） | 记录次数，无即时惩罚 |
| 严重碰撞 | 车辆间高速碰撞（相对速度 >= 3 m/s） | 触发停车 2 秒（当前实现中车辆会永久静止，见下方已知问题） |
| 取消资格 | 累计 3 次严重碰撞 | 本场停止行驶，状态标记为 `disqualified` |
| 滞留违规 | 60 秒内未通过任何检查点 | 取消资格 |

### 罚停机制（已知问题）

- Supervisor 在检测到严重碰撞后，会向车辆发送 `stop` 指令，并在 2 秒后清除该指令、恢复 `normal` 状态
- **当前版本限制**：`car_controller.py` 收到 `stop` 指令后内部状态被置为永久停止，即使 Supervisor 清除指令，车辆也不会恢复行驶。该问题将在后续版本修复

## 5. 代码沙箱限制

详见 `README_code_submission_guide.md`。

核心限制：
- 禁止导入系统/网络/文件模块（`os`, `sys`, `socket`, `threading` 等）
- 禁止 `open`, `eval`, `exec`, `globals`, `locals`, `compile`
- Linux 下限制内存 512 MB、CPU 时间 30 秒

## 6. 状态机流转

每个赛区独立维护状态机（`server/race/state_machine.py`）：

```
REGISTRATION --[lock]--> IDLE --[start-race]--> PLACEMENT_RUNNING
                                                          |
                    |---------------------------------------|
                    v
           PLACEMENT_FINISHED / PLACEMENT_ABORTED
                    |
                    v
           PLACEMENT_DONE --[finalize]--> GROUP_STAGE_RUNNING --> GROUP_STAGE_DONE
                    |                                                    |
                    |----------------------- SEMI_RUNNING --> SEMI_DONE --|
                                                                |
                                                        FINAL_RUNNING --> FINAL_FINISHED --> CLOSED
```

- `REGISTRATION`：允许上传代码
- `IDLE`：待命，可配置/启动比赛
- `*_RUNNING`：仿真进行中，测试队列暂停消费
- `*_FINISHED`：单场比赛结束，等待 finalize
- `finalize`：自动计算下一阶段对阵并预创建场次
- `reset`：从任意状态强制回到 `IDLE`

## 7. 积分规则

仅向**完赛**车辆（`total_time is not None`）发放名次积分：

| 名次 | 积分 |
|------|------|
| 第 1 名 | 10 分 |
| 第 2 名 | 7 分 |
| 第 3 名 | 5 分 |
| 第 4 名 | 3 分 |
| 第 5 名及以后 / 未完赛 | 1 分 |

总积分榜按 `SUM(points)` 降序排列，同分时可参考 `MIN(best_lap_time)`。
